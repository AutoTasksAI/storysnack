⚙️ Setup

Step 1 — Story Generation (required)

StorySnack uses Claude AI to write stories. You need a free Anthropic API key.

👉 Get your key at console.anthropic.com — sign up free, go to API Keys, create one, paste it here.

Step 2 — Audio Narration (optional)

Want the story read aloud in a warm storyteller voice? Add a Google AI Studio key.

👉 Get your key at aistudio.google.com/apikey — free tier included. Click "Create API key."

🔒 Your keys are saved only in this browser. They are never sent to anyone except Anthropic and Google directly. We do not store anything on a server.
